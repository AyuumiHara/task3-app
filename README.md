# 🍎 Task-app
 
家庭にあるものの期限管理アプリ
 
家庭で使っている、サブスクや、防虫剤の期限を管理し、最終日が近づいてきたらお知らせ機能付き


# 🌿 DEMO
 
![スクリーンショット 2021-06-11 22 57 05](https://user-images.githubusercontent.com/77601536/121698456-fb3a2200-cb08-11eb-8739-3691380d8be4.png)
 
 
# 🌿 Features
 
* ユーザ登録して家族で共有できる
* サブスク、防虫剤、購読雑誌等でカテゴリ分けできる
